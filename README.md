WeatherScope is a weather dashboard where users can register, log in, and search any city worldwide. The backend fetches data from 3 APIs Geoapify converts the city name to coordinates, OpenWeatherMap returns the weather, and Unsplash provides a background image. The UI dynamically changes its color theme and animations based on the weather condition and temperature. Users can save favorite cities, view a 5-day forecast chart, and explore nearby cities. The entire stack uses Django REST Framework, React with Vite, and PostgreSQL.
